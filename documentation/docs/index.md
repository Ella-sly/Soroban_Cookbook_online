Your comprehensive guide to building smart contracts on Stellar.

## 🚀 What is Soroban?

Soroban is Stellar's smart contract platform that enables developers to write and deploy smart contracts using Rust. Built for scalability, security, and developer experience.

## 📚 What You'll Learn

Build production-ready smart contracts on Soroban, from basic concepts to advanced patterns:

- **Environment Setup** - Get your development environment ready
- **Core Concepts** - Understand Soroban's architecture and patterns
- **Smart Contracts** - Build real-world contract examples
- **Best Practices** - Learn security and optimization techniques

## ✨ Features of Soroban

- ⚡ **Sub-second finality** - Fast transaction confirmation
- 💰 **Low costs** - Minimal transaction fees
- 🔒 **Secure** - Built with Rust's memory safety
- 🦀 **Rust-powered** - Leverage Rust's ecosystem
- 🌐 **Stellar network** - Access to global financial infrastructure
